### 🚀 Projects (Selected Works)

#### 1. [喫煙所マップ](https://github.com/twatatani/smoking-map.git)
🗺️ 喫煙所マップ管理アプリ
管理者が喫煙所情報を登録・編集・削除できる Web アプリケーションです。
現在地取得や住所の自動入力など、スマートフォンからの操作を想定した UI を備えています。
📌 概要
このアプリは、街中の喫煙所を効率的に管理するための管理者向けツールです。
名称・住所・緯度・経度・説明文などを入力して喫煙所を登録できます。
スマートフォンからでも直感的に操作できるよう、
レスポンシブデザインや位置情報 API を活用して使いやすさを重視しています。
🛠 使用技術
フロントエンド
・HTML / CSS / JavaScript
・Leaflet.js（地図表示）
・Geolocation API（現在地取得）
バックエンド
・Java
・Spring Boot
・Thymeleaf（テンプレートエンジン）
データベース
・postgreSQL（または使用している DB に合わせて変更）
API
・Nominatim API（OpenStreetMap）
　→ 緯度・経度から住所を自動取得
開発環境・その他
・Eclipse
・Git / GitHub
